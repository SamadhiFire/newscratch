## Generation Input

`scripts/score-and-select.ps1` writes `processed/generation-input.json`. Each item contains selected source material and scoring metadata:

```json
[
  {
    "category": "科技AI",
    "categoryKey": "tech",
    "categoryEnglish": "Technology and AI",
    "sourceUrl": "https://example.com/news/1",
    "sourceTitle": "Original source title",
    "sourceDescription": "Original source description",
    "sourceBody": "Original source body",
    "sourceImage": "https://example.com/source.jpg",
    "sourceName": "Example News",
    "publishedAt": "2026-05-20 09:30:00",
    "score": 24,
    "evaluation": "相关性8, 新颖性8, 完整度8. 通过原因: ...",
    "status": "待生成",
    "publishStatus": "未发布"
  }
]
```

The model must transform that input into `records.normalized.json`.

## Normalized Output

`scripts/write_lark_records.ps1` expects `records.normalized.json` to be a JSON array:

```json
[
  {
    "category": "科技AI",
    "sourceUrl": "https://example.com/news/1",
    "sourceTitle": "Original source title",
    "sourceBody": "Original source body or description",
    "publishedAt": "2026-05-20 09:30:00",
    "generatedTitle": "AI Data Center Expansion Puts New Pressure on Power Supply",
    "body": "A formal English news article generated by the model.",
    "imagePrompt": "Professional editorial cover image showing...",
    "generatedImageUrl": "https://example.com/generated-cover.png",
    "generatedImagePath": "C:\\path\\to\\generated-cover.webp",
    "generatedBy": "model",
    "score": 24,
    "evaluation": "相关性8, 新颖性8, 完整度8. 通过原因: ...",
    "status": "已生成",
    "publishStatus": "未发布"
  }
]
```

Minimum required fields:

- `category`
- `sourceUrl`
- `generatedTitle`
- `body`
- `imagePrompt`
- `generatedBy` with value `model`
- `score`

Recommended preserved fields:

- `sourceTitle`
- `sourceBody`
- `publishedAt`
- `evaluation`
- `status`
- `publishStatus`
- `generatedImageUrl`
- `generatedImagePath`

## Model Output Rules

- `generatedTitle`: complete English headline, 45-90 characters, not copied blindly from `sourceTitle`, and not ending with `...`.
- `body`: formal English news article, 700-900 words (approximately 3,500-5,000 characters), written for readers.
- `body` must not begin with `A report from`, `According to the source`, or `This story fits`.
- Do not include raw source truncation suffixes like `[+1433 chars]`.
- Do not invent facts, quotes, numbers, places, dates, or outcomes unsupported by the source fields.
- `imagePrompt`: 16:9 editorial cover prompt, no text, watermark, or logo.
- `generatedImageUrl`: optional remote URL created by the image API; if present it must be an `http/https` URL.
- `generatedImagePath`: optional local file path created by the script, typically a `.webp` file for later Feishu attachment upload.
- `generatedBy`: must be exactly `model`.
